# 🍔 Al Baik Fast Food Website

A responsive fast food website inspired by Al Baik. Built using HTML, CSS, and JavaScript, showcasing menu categories like chicken burgers, veg burgers, grills, and sandwiches with a clean and modern UI.

---

## 🚀 Features
- Responsive design (mobile + desktop)
- Organized menu sections
- Simple and clean UI
- Fast loading static website

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
